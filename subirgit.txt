1. Inicializar Git en tu proyecto
Abre tu proyecto en Visual Studio Code.

Accede a la terminal integrada (Ctrl + \ o Cmd + \ en Mac).

Inicializa Git si aún no lo has hecho:

bash
Copiar
Editar
git init
Añade los archivos del proyecto al área de preparación (staging):

bash
Copiar
Editar
git add .
Crea un commit inicial con un mensaje descriptivo:

bash
Copiar
Editar
git commit -m "Commit inicial"
2. Crear un repositorio remoto en GitHub
Ve a tu cuenta de GitHub y haz clic en el botón New Repository.
Asigna un nombre al repositorio (opcionalmente agrega una descripción).
Configura el repositorio como público o privado.
No marques la opción de inicializar con un README (ya tienes un commit inicial).
Haz clic en Create repository.
Copia la URL del repositorio remoto.
3. Vincular el repositorio remoto
Regresa a Visual Studio Code y en la terminal integrada, ejecuta:

bash
Copiar
Editar
git remote add origin https://github.com/vazperale/OnlineShop.git
Reemplaza <URL_del_repositorio> con la URL copiada del paso anterior (por ejemplo: https://github.com/usuario/nombre-del-repositorio.git).

4. Subir el proyecto al repositorio remoto
Asegúrate de estar en la rama principal (generalmente main o master):

bash
Copiar
Editar
git branch -M main
Sube los cambios al repositorio remoto:

bash
Copiar
Editar
git push -u origin main
